# ESTP 前额叶训练营 ⚡

一个为 ESTP 设计的纯前端网页应用,专门训练**专注力、自控力、冲动抑制**——也就是大脑前额叶皮层最核心的几个执行功能。

## 为什么是 ESTP

ESTP 大脑像 V8 引擎:Se 主导让感官输入异常活跃、爱即时反馈、容易冲动。短板恰恰在前额叶执行功能(规划、抑制、延迟满足)。这个 App 把前额叶训练做成 ESTP 喜欢的形式——**短回合 + 即时数据 + 等级系统 + 可见的连胜**。

## 训练模块

| 模块 | 训练的能力 | 科学依据 |
|---|---|---|
| 🍅 专注番茄钟 | 持续性注意 | Pomodoro 技术 |
| 🧠 N-Back | 工作记忆 | Jaeggi et al., 2008 |
| 🌈 Stroop 测试 | 冲突抑制 | 经典前额叶任务 |
| 🚦 Go / No-Go | 冲动控制 | 反应抑制范式 |
| 🌬️ 4-7-8 呼吸 | 副交感激活、情绪调节 | Andrew Weil 法 |

每完成一次训练都会得分,累计分数升级,并在仪表盘上看到 14 天热力图、连续打卡、各模块的最佳成绩。

## 怎么用

直接用浏览器打开 `index.html` 即可,**无需任何构建步骤、无需服务器、无需联网**。

```bash
# 任选一种
open index.html              # macOS
xdg-open index.html          # Linux
start index.html             # Windows
```

或者用任意静态服务器:

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

## 数据隐私

所有进度保存在浏览器的 `localStorage`,**不上传任何服务器**。可以在仪表盘随时导出 JSON 备份或重置进度。

## 建议训练计划(给你抄作业)

**第 1 周:每天 10 分钟,只做 3 件**
- 5 分钟番茄钟 × 1
- 1-Back 一局
- 4-7-8 呼吸 3 轮

**第 2-4 周:每天 15 分钟,加难度**
- 15 分钟番茄钟 × 1
- 2-Back 一局
- Stroop 一局
- 呼吸 4 轮

**第 5 周以后:每天 20 分钟,全模块覆盖**
- 25 分钟番茄钟 × 1
- 3-Back / Go-NoGo / Stroop 各一局
- 呼吸收尾

坚持到第 90 天,你应该能感觉到:做事不再三分钟热度、看到手机不会立刻拿、说话之前能停顿半秒。
